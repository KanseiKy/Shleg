# Shleg
🐚 A repo containing games written purely in Shell

# Getting started
* Clone this repo
```sh
$ git clone https://github.com/KanseiKy/Shleg.git
$ cd git
```

* Choose a game to play
```sh
$ bash game_[n].sh
```
> `n` is the number of the game

* Enjoy
